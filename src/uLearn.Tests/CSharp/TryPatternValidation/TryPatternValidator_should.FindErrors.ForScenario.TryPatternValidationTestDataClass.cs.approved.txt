Строка 5, позиция 2: Допустим только один out параметр в сигнатуре метода
Строка 12, позиция 2: Недопустимо использовать ref параметры, разрешено использовать только 1 out параметр
Строка 18, позиция 2: Название метода не начинается с Try
Строка 24, позиция 2: Название метода не начинается с Try
Строка 24, позиция 2: Метод не возвращает bool
Строка 29, позиция 2: Метод не возвращает bool
Строка 34, позиция 2: Out параметр не является последним аргументом
Строка 40, позиция 2: Недопустимо использовать ref параметры, разрешено использовать только 1 out параметр
